# CSV to JSON Demo

This is a live demo of the CSV to JSON converter library.

## Features

- Convert CSV text to JSON
- Upload CSV files
- Configure parsing options:
  - Format values by type
  - Support quoted fields
  - Custom field delimiter
  - Header row index

## Usage

1. Choose between pasting CSV text or uploading a file
2. Configure options as needed
3. Click "Convert to JSON" to see the result

## Deployment

This demo is deployed using GitHub Pages from the `docs/` folder.
